# Schema Definitions

## Overview

The analytical data warehouse follows a Star Schema design consisting of four dimension tables and four fact tables. The model supports dashboarding, historical reporting, and machine learning workloads.

---

# Dimension Tables

## 1. dim_site

Stores site-level information.

| Column | Description |
|---------|-------------|
| site_id | Primary Key |
| customer_id | Customer identifier |
| customer_name | Customer name |
| industry | Industry type |
| site_name | Site name |
| city | City |
| state | State |
| country | Country |
| site_type | Site category |

---

## 2. dim_building

Stores building information for each site.

| Column | Description |
|---------|-------------|
| building_id | Primary Key |
| site_id | Foreign Key → dim_site |
| building_name | Building name |

---

## 3. dim_asset

Stores asset metadata.

| Column | Description |
|---------|-------------|
| asset_id | Primary Key |
| site_id | Foreign Key → dim_site |
| building_id | Foreign Key → dim_building |
| customer_id | Customer identifier |
| asset_type | Asset type |
| asset_name | Asset name |
| category | Asset category |
| manufacturer | Manufacturer |
| installation_date | Installation date |
| warranty_expiry | Warranty expiry |

---

## 4. dim_time

Stores time attributes.

| Column | Description |
|---------|-------------|
| time_id | Primary Key |
| full_date | Timestamp |
| hour | Hour |
| day | Day |
| month | Month |
| quarter | Quarter |
| year | Year |
| weekday | Weekday |

---

# Fact Tables

## 1. fact_telemetry

Stores IoT sensor readings.

Measures:

- Temperature
- Humidity
- Pressure
- Vibration
- Power Consumption
- Operating Mode

Foreign Keys

- time_id
- asset_id

---

## 2. fact_event

Stores operational events generated by assets.

Measures

- Event Type
- Severity
- Message

Foreign Keys

- time_id
- asset_id

---

## 3. fact_energy

Stores hourly energy consumption metrics.

Measures

- Hourly Energy
- Average Power

Foreign Keys

- time_id
- asset_id

---

## 4. fact_asset_health

Stores asset health indicators.

Measures

- Health Score
- Risk Level

Foreign Keys

- time_id
- asset_id

---

dim_site

- Star Schema
